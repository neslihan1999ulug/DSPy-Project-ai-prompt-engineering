# DSPy AI Prompt Engineering

Learning modern AI development with DSPy — a framework that replaces manual prompt engineering with automated, optimized LLM pipelines.

## What is DSPy?

Instead of writing prompts manually, DSPy acts as a **compiler** for your AI programs. You define _what_ you want, DSPy figures out _how_ to prompt the model.
